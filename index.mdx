## What is Lidraughts?

Lidraughts is a fully open-source, ad-free online draughts platform — a rewrite of [lichess/lila](https://github.com/ornicar/lila/) adapted for draughts. It supports multiple board sizes and rule variants, from the classic 10×10 international draughts to compact 8×8 variants.

All features are free for everyone, forever. No ads, no premium paywalls, no subscriptions required to play.

## Supported variants

Lidraughts supports seven distinct draughts variants across two board sizes:

| Variant | Board | Description |
|---|---|---|
| Standard | 10×10 | International draughts — the classic game |
| Frisian | 10×10 | Pieces can capture horizontally and vertically |
| Antidraughts | 10×10 | The goal is to lose all your pieces |
| Frysk! | 10×10 | Each player starts with only 5 pieces |
| Breakthrough | 10×10 | First to reach the back rank wins |
| Russian | 8×8 | Russian draughts with flying kings |
| Brazilian | 8×8 | Brazilian rules on 8×8 board |
